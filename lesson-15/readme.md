# Lesson 15

In this lesson we explored css and xpath selectors.


## Getting Started

Selectors are specified in the `tests/selectors.spec.ts` file.

To install the necessary dependencies, run:
```
npm i
```


To run the tests, you can use Node.js.
```
npm run test
```


To run the tests in headed mode, use:
```
npm run test:headed
```

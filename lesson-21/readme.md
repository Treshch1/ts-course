# Lesson 21

In this lesson we explored reports.


## Getting Started

To install the necessary dependencies, run:
```
npm i
```


To run the tests, you can use Node.js.
```
npm run test
```

To see allure report install allure and run:
```
allure serve allure-results
```

To see mochawesome report, open the `mochawesome-report/mochawesome.html` file in your browser.
